Debug your LangGraph agent.

A tiny, open-source lib to interact with your langgraph agent and visualize its execution.

$ npm install agent-debug

Quick Start

Install agent-debug package via npm:

npm install agent-debug

Install langgraph package via npm:

npm install @langchain/langgraph

Expose your LangGraph instance for debugging purposes:

import { debug } from 'agent-debug';
import { StateGraph, START, END, MessagesAnnotation } from "@langchain/langgraph";

const workflow = new StateGraph(MessagesAnnotation)
  .addNode("agent", agentNode)
  .addEdge(START, "agent")
  .addEdge("agent", END);

const agent = workflow.compile();

// spinup server and open agent-debug interface
debug(agent, 3001)
