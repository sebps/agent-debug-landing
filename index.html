Quick Start

1. Expose your LangGraph instance via our helper in your server.js:

    app.get('/graph', async (req, res) => {
    const structure = await graph.getGraphAsync();
    res.json(structure);
    });

2. Launch the debugger UI and point it to your local port:

VITE_AGENT_URL=http://localhost:3000 npm start

A minimalist tool for the LangGraph community. No tracking. Just debugging.
